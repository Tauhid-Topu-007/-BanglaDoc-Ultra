# BanglaDoc Ultra — বাংলা ডকুমেন্ট প্রসেসিং আলট্রা
FastAPI-based AI platform for advanced Bangla document processing  
**Version: 3.0**

---

## ✨ Overview
BanglaDoc Ultra is an AI-powered Bengali document processing API built with FastAPI.  
It includes Bangla NLP utilities, smart word suggestions, idiom detection, Bangla calendar conversion, text analysis, and more.

---

## 🔥 Key Features

### 📘 Bangla NLP
- বাংলা শব্দ যাচাই
- নিকট শব্দ সাজেশন (similar word matching)
- শব্দভাণ্ডার ক্যাটাগরি সনাক্তকরণ
- বাংলা বাগধারা ও অর্থ
- ইউনিকোড স্বাভাবিককরণ (Normalization)

### 📅 Bangla Calendar System
- বাংলা বছর, মাস, দিন হিসাব
- ইংরেজি → বাংলা সংখ্যা রূপান্তর
- দিনের বাংলা নাম
- ঋতু নির্ণয়

### 📄 Document Processing
- টেক্সট আপলোড ও প্রসেসিং
- শব্দ গণনা
- বাংলা শব্দ ফ্রিকোয়েন্সি বিশ্লেষণ
- ডুপ্লিকেট ও নন-বাংলা ফিল্টারিং

### 🛠 Tech Features
- FastAPI auto documentation (Swagger / Redoc)
- Full CORS support
- JSON-based API responses
